# GAON
### 📁 작업 규칙

1. **개별 폴더에서 작업 시, 폴더 구조는 `Main`과 최대한 동일하게 맞추기**  
   → 병합 시 충돌 없이 쉽게 통합할 수 있도록 하기 위함

2. **파일 이름 규칙 (접두사 붙이기)**  
   → 역할별로 빠르게 식별하고 정리하기 위함

   | 접두사   | 설명                           |
   |----------|--------------------------------|
   | `M_`     | 머티리얼 (Material)            |
   | `T_`     | 텍스쳐 (Texture)               |
   | `AC_`    | 애니메이션 컨트롤러 (Animator Controller) |
   | `Anim_`  | 애니메이션 클립 (Animation Clip) |
   | `P_`     | 프리팹 (Prefab)                |
